# CS-330

Throughout this project, I approached software design by breaking the 3D scene into modular helper functions, such as separate methods for rendering platforms and characters. This made the code cleaner and easier to maintain. I followed an iterative process, starting with basic elements like the ground and lighting, then gradually adding more complex objects while frequently testing changes.

My development strategy relied on building the scene incrementally and testing after nearly every change. Iteration played a major role, especially when fine tuning character shapes, rotations, and materials. Over the course of the milestones, my approach evolved from following the sample code closely to confidently writing my own custom transformations and scene logic.

Working with computational graphics strengthened my understanding of linear algebra, spatial reasoning, and real time rendering. These skills will support my future coursework in areas like game development and data visualization. Professionally, they enhance my ability to work on interactive 3D applications, virtual reality, and engaging visual projects.
